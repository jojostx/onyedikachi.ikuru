Cyclistic Bike-Share Analysis

Google Data Analytics Capstone | 2024
Case Study R Python

Comprehensive data analysis of Cyclistic bike-share usage patterns to identify behavioral differences between annual members and casual riders. The project addresses a key business question: how to convert casual riders into profitable annual members through targeted marketing strategies.

Analyzed 12 months of ride data (2024) encompassing millions of trips, uncovering distinct usage patterns across temporal, spatial, and behavioral dimensions.

Key Findings

  • Members demonstrate commuter-focused patterns with weekday peaks during rush hours (7-9 AM, 4-6 PM)
  • Casual riders show recreational patterns with weekend peaks and longer trip durations
  • Electric bikes preferred by casual riders, classic bikes by members
  • Distinct station preferences indicate commuting vs. tourist/recreational usage
  • Strong seasonal trends for casual riders, consistent usage for members

Deliverables

  • Complete data cleaning and processing pipeline handling millions of records
  • Statistical analysis across temporal, behavioral, and spatial dimensions
  • 10+ data visualizations (bar charts, line graphs, ridge plots, heatmaps)
  • Station classification system using Google Gemini API
  • 5 targeted marketing strategies with implementation roadmaps
  • Comprehensive R Markdown report following case study methodology
